Heliactyl Documentation

Dependencies

Before installing Heliactyl, make sure you have the following dependencies installed on your system:


Ubuntu / Debian

# Updating the VPS
sudo apt update
# Installing Nodejs
sudo apt install nodejs
sudo apt install npm 
# Installing git
sudo apt install git

Installation

The following commands will download the Heliactyl Dashboard into /etc/heliactyl and use npm to install the required packages:


cd /etc
git clone https://github.com/heliactyloss/heliactyl
cd heliactyl
npm install

Setup Complete

All you need to do now is start Heliactyl:


node .

Configure

Now you can go in settings.json Where you Can add Pterodactyl Panel Domain and ptla_key****** / Discord Auth Settings
